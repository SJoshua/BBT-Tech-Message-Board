# BBT-Tech-Message-Board
这是2018级[百步梯](https://www.100steps.net/)技术部的第三次及第四次部门任务。

## Features
* 用户操作 (注册、登录与注销)
* 留言操作 (发送、修改与删除)
* 对留言的评论

## To-do
* 头像

## Configuration
1. 以 `config-sample.php` 为例，在 `config.php` 中填写数据库名及连接凭证。
2. 运行 `install.php` 以创建保存用户信息及留言数据的表。
